# Chat App

A real-time chat application built with **React**, **Node.js**, **Express**, **MongoDB**, and **Socket.io**. Users can sign up, login, chat with others, send images, see who is online, and manage their profile.

---

## Features

- **User Authentication**: Signup, Login, and Profile update.
- **Real-Time Chat**: Instant messaging using Socket.io.
- **Image Sharing**: Send images in chat.
- **Online Status**: See which users are online.
- **Unread Messages**: Shows count of unseen messages.
- **Profile Management**: Update name, bio, and profile picture.
- **Responsive UI**: Works on both desktop and mobile.

---

## Technologies Used

- **Frontend**: React, Tailwind CSS, React Router
- **Backend**: Node.js, Express
- **Database**: MongoDB, Mongoose
- **Realtime**: Socket.io
- **Image Storage**: Cloudinary
- **Authentication**: JWT (JSON Web Tokens)
- **Notifications**: react-hot-toast
